# MindNest

## Setup
1. Install MongoDB and run it locally.
2. In `/server`, run `npm install` then `node server.js`.
3. In `/client`, run `npm install` then `npm start`.
4. Visit http://localhost:3000 for the frontend.

## Features
- Student login/signup
- Post and read blogs
- Profile dashboard

To deploy on GitHub:
- Create two repos (client & server) and push each folder.
- Optionally, merge into one repo with `/client` and `/server` directories.

For production: Consider a cloud MongoDB and secure keys/URLs.
